🔧 Firebase 환경변수 설정 필요

Railway 환경변수에 Firebase 설정값을 입력하세요.

【Railway 환경변수】
FIREBASE_API_KEY            → Firebase 프로젝트 설정의 apiKey
FIREBASE_AUTH_DOMAIN        → authDomain
FIREBASE_PROJECT_ID         → projectId
FIREBASE_STORAGE_BUCKET     → storageBucket
FIREBASE_MESSAGING_SENDER_ID → messagingSenderId
FIREBASE_APP_ID             → appId
OPENAI_API_KEY              → OpenAI API 키 (AI챗봇/변호사 사용시)
EMAILJS_PUBLIC_KEY          → EmailJS (자동메일 선택)
EMAILJS_SERVICE_ID
EMAILJS_TEMPLATE_ID

【Firebase 프로젝트 생성】
1. https://console.firebase.google.com → 새 프로젝트
2. Authentication → Google 로그인 활성화
3. Firestore → 테스트 모드로 생성
4. 프로젝트 설정 → 웹앱(</>) 등록 → firebaseConfig 복사
